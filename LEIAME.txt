# MANUAL DE OPERAÇÃO - SCRIPTS DE MANUTENÇÃO
# DELÍCIA DA CIDADE - KITCHEN OS

Este arquivo explica como utilizar os scripts automáticos criados para facilitar a manutenção, backup e atualização do sistema.

----------------------------------------------------------------------
1. BACKUP LOCAL (backup_local.bat)
----------------------------------------------------------------------
O QUE FAZ:
Cria uma cópia completa e segura de todo o projeto para uma pasta externa chamada "Backups_DeliciaDaCidade".
Cada backup é salvo em uma pasta com a Data e Hora atual, evitando sobrescrever backups antigos.

COMO USAR:
1. Vá até a pasta do projeto.
2. Dê um duplo clique no arquivo "backup_local.bat".
3. Aguarde a mensagem "BACKUP CONCLUIDO COM SUCESSO!".
4. Pressione qualquer tecla para fechar.

ONDE FICAM OS BACKUPS:
Eles são salvos na pasta "Backups_DeliciaDaCidade" localizada um nível acima da pasta do projeto.

----------------------------------------------------------------------
2. RESTAURAÇÃO DE SISTEMA (restore_local.bat)
----------------------------------------------------------------------
O QUE FAZ:
Permite recuperar o sistema para uma versão anterior salva pelo script de backup. Útil caso algo pare de funcionar após uma alteração.

COMO USAR:
1. Dê um duplo clique no arquivo "restore_local.bat".
2. O script listará todos os backups disponíveis numerados.
3. Digite o número do backup que deseja restaurar e pressione ENTER.
4. O script substituirá os arquivos atuais pelos arquivos do backup escolhido.

ATENÇÃO: Isso sobrescreve as alterações atuais! Faça um backup antes se não tiver certeza.

----------------------------------------------------------------------
3. ATUALIZAÇÃO NO SERVIDOR (atualizar.bat)
----------------------------------------------------------------------
O QUE FAZ:
Envia as alterações feitas no código para o GitHub/Netlify, atualizando o sistema online.

COMO USAR:
1. Após fazer alterações no código, dê um duplo clique em "atualizar.bat".
2. O script fará o processo de envio automaticamente.
3. Aguarde a mensagem de sucesso.

----------------------------------------------------------------------
4. SEGURANÇA E BACKUP DO BANCO DE DADOS (Supabase)
----------------------------------------------------------------------
O arquivo "seguranca_e_backup.sql" contém comandos para proteger seus dados.

COMO USAR:
1. Abra o arquivo "seguranca_e_backup.sql" no VS Code ou Bloco de Notas.
2. Copie o código desejado (ex: Parte 2 para criar cópia dos dados).
3. Cole no "SQL Editor" dentro do painel do seu projeto no Supabase.
4. Clique em "Run" para executar.

----------------------------------------------------------------------
5. AGENDAMENTO AUTOMÁTICO NO SUPABASE (agendamento_backup.sql)
----------------------------------------------------------------------
Este arquivo permite configurar o Supabase para criar cópias das tabelas automaticamente todo dia.

COMO USAR:
1. Ative a extensão "pg_cron" no painel do Supabase (Database > Extensions).
2. Rode o script "agendamento_backup.sql" no SQL Editor.
3. O sistema criará tabelas como "Backup_Estoque_2023_..." automaticamente toda madrugada.

----------------------------------------------------------------------
6. TESTE E DIAGNÓSTICO AUTOMÁTICO (testar_sistema.bat)
----------------------------------------------------------------------
Ferramenta para verificar se existem erros de código ou arquivos faltando antes de atualizar.

COMO USAR:
1. Dê um duplo clique em "testar_sistema.bat".
2. O script verificará a sintaxe de todos os arquivos JavaScript.
3. Verificará se todos os links (CSS, JS, Páginas) nos arquivos HTML estão corretos.
4. Exibirá um relatório de saúde do sistema.

----------------------------------------------------------------------
7. MIGRAÇÃO DE DADOS (migracao_dados.sql)
----------------------------------------------------------------------
Contém modelos de scripts SQL para transferir dados entre tabelas.

COMO USAR:
1. Abra "migracao_dados.sql".
2. Copie o exemplo que se adapta à sua necessidade.
3. Ajuste os nomes das tabelas e colunas.
4. Execute no SQL Editor do Supabase.
